# Getting Started

This section takes you from zero to a running lesson. By the end you'll have a working HTML file that plays narration, renders notebook cards, and drives a synchronized visualization.

## Prerequisites

- A terminal (macOS/Linux; Windows with WSL works too)
- Python 3 (for the build script — no install needed)
- A modern browser (Chrome, Firefox, Safari, Edge)
- A text editor

No npm install, no webpack, no bundler. The build script is a single Python one-liner inlined into `build.sh`.
